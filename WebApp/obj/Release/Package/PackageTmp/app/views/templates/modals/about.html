Site Technology

This site utilizes a few differnt pieces of new tecnology. Although majority is done in JavaScript, some of the frameworks that are used are relatively new

 1. AngularJs
AngularJs is a new JavaScript framework (supported by Google) that allows to make very dymanic web page.
Key Features:
  • Two Way Binding.
  • Use of Templates to allow for Single Page Applications (SPA).
  • Dependency Injetion in code.
  • Directives - allow to create custom HTML tags with custom functionality
  • Its MVC, or MVVM or like Google says MVW where W means 'Whatever' just depends on how you use it.
  • Built in compatibilty for Unit Testing - Greate for TDD
  • Great Bootstrap UI compatibilty
  • Great Documentation and community

 2. Bootstrap
Bootstrap is also a new framework, it is primarliy used for presentation. The library includes both CSS and JavaScript files for creating a very modern UI to provide a rich user experience.
Key Features:
  • Provides a great base for all HTML pages for styling
  • Get started quickly - has everything you need set up already in the include files
  • Great starting place for web developers to use (those that aren't as artistic as designers)
  • Lots of plug-ins
  • jQuery Support
  • Plays well with AngularJs
  • Greate documentation and community

 3. TypeScript
TypeScript is a free and open source technology created and maintained by the Microsoft® Corporation.
TypeScript was created by Microsoft, not as a replacement or enhancment to JavaScript, but as a help bring greater structure and managability to JavaScript for developers.
With type script you can create classes and other code structures to help you manage your code better. When TypeScript is compiled the result is a .js JavaScript file, so the .ts / TypeScript files are only utilized when creating code and compiling it.
Key Features:
  • Provides a great base for all HTML pages for styling
  • Get started quickly - has everything you need set up already in the include files
  • Great starting place for web developers to use (those that aren't as artistic as designers)
  • Lots of plug-ins
  • jQuery Support
  • Plays well with AngularJs
  • Greate documentation and community
