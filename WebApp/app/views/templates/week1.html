Week 1: Semantic Tags

This week we are dealing with using semantic tags in an HTML5 Web Application

What are Semantic Tags

Semantic tags are typically tags that have a name that gives a good description of the tag's contents.

Examples:
<table>, <headder>, <footer>, and <article> are some examples. Each of these tags have names that discribe exactly what they do.
Very useful for developers and designers.
